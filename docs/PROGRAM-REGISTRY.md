## Status definitions

- **Concept:** Idea exists but scope and acceptance criteria are incomplete.
- **Planned:** Purpose, owner, and next milestone are defined.
- **Active:** Work is underway with current evidence or changes.
- **Blocked:** Progress requires a decision, access, dependency, or repair.
- **Review:** Deliverable exists and awaits verification or approval.
- **Operational:** Repeatable workflow is in use and maintained.
- **Paused:** Intentionally deferred with preserved context.

## Intake rule

New ideas should not automatically become standalone projects. First determine whether the idea belongs inside an existing program. Create a new program only when it has a distinct user, workflow, data boundary, and measurable outcome.
